# Roadmap

This document outlines the future development plans for Garhwali Seva.

## Vision

To become the definitive platform for Garhwali language preservation, education, and community engagement.

---

## Version 1.0 (Current)

**Status**: ✅ Complete

### Features Delivered

- [x] Basic text translation (Garhwali → Hindi/English)
- [x] Song lyrics translator
- [x] Speech-to-text input
- [x] Community contribution form
- [x] Basic dictionary (~150 words)
- [x] Responsive design
- [x] Dark/light mode
- [x] Vercel deployment ready

## Contributing to the Roadmap

We welcome community input on our roadmap!

### How to Contribute

1. **Feature Requests**: Open a [GitHub Issue](https://github.com/yourusername/garhwali-seva/issues) with "feature-request" tag
2. **Priority Discussion**: Comment on [Roadmap Discussions](https://github.com/yourusername/garhwali-seva/discussions/categories/roadmap)
3. **Implementation**: Submit PRs for roadmap features

### Community Voting

Upcoming features are voted on by the community:
- Top-voted features get priority
- Voting opens at the start of each quarter
- Results announced in release notes
